[改良版] ウェブサイトやブログにCiNii論文情報を挿入できるjQueryプラグイン

前回のNII論文ID(NAID)からCiNiiの論文情報を取得するjQueryのプラグインの別バージョンです。

前回のに対してコメントを頂いたのを参考に作り変えました。

ダウンロード サンプルページ

まず下の通りheadにプラグインの設定をします。

<script type="text/javascript">
  $(document).ready(function(){
	$("div[data-naid]").getCiNiiDataNaid();
  });
</script>

次に論文情報を表示させるdiv要素のdata属性(data-naid)にカンマ区切りでnaidを入力するだけです。

現在、配列の順番に展開されないことがあります。
<div data-naid="110009040355,40019076036,110008916540"></div>

今回のほうがいいですね。 サンプルページみたいに1ページの中で数箇所に設置するのも簡単です。
