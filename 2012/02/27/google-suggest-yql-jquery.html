Google SuggestをYQLを使って取得するjQueryプラグイン

Google Suggest APIは現在xml形式のみなので、YQLでJSONPに変換してjQueryで取得するプラグインを作ってみました。Google SuggestはGoogle検索時に単語候補予測をしてくれるやつですね。

OPACなどでヒット件数0件の時とかに他のキーワード候補として使えるかもしれませんね。あと、スペスミスのときに、「もしかして・・・」と検索結果のページに出したり。ただ、芸能人の名前とか出てくることもありますが…

ダウンロード サンプルページ

使い方

まずは、ヘッダにjQueryと本プラグインを読み込みます。

<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js"></script>
<script type="text/javascript" src="jquery.google.suggestion.js"></script>

Google Suggestの結果を表示させたい要素のidを設定して、オプションのkeywordを入力してください。

<script type="text/javascript">
  // 例）sample01というid要素にキーワード「google」の結果を表示
  $(document).ready(function(){
	$("#sample01").getGoogleSuggestion({keyword:"google"});
  });
</script>

リンク付きでリスト表示されます。

デフォルトではGoogleへのリンクが貼られています。変更する場合は、検索サイトにあわせてプラグインの下記の部分（37行目あたり）を変えてください。

html += "<li><a href='http://www.google.co.jp/search?&q=" + item[i].suggestion.data + "' target='_blank'>" + item[i].suggestion.data + "</a></li>";

下のように検索フォームでキーワードを取得するなど使い方は色々あると思います。

「gooogle」と間違ったスペルでも候補を返してくれます。
