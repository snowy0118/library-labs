人名から生没年、生没年から人名をWeb NDL Authoritiesから取得する

Web NDL AuthoritiesのSPARQLによる検索で、人名から生没年、生没年から人名を調べることができます。今回はその検索結果をjQueryで取得する2つのプラグインです。いつものようにYQLでJSONをJSONPに変換することで別のドメインにあるJSONデータをjQueryでロードしています。

ダウンロード サンプルページ

人名から生没年を得るプラグイン

まずは、ヘッダにjQueryとプラグイン(jquery.ndla.person.name.js)を読み込みます。

<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js"></script>
<script type="text/javascript" src="jquery.ndla.person.name.js"></script>

検索結果を表示させたい要素のidを設定して、オプションのnameを入力してください。

<script type="text/javascript">
  // 例）sample01というid要素に人名「中上健次」の生没年を表示
  $(document).ready(function(){
	$("#sample01").getNdlaFromName({person:"中上健次"});
  });
</script>

下のように読みと一緒に生没年が表示されます。

生没年から人名

同じく、ヘッダにjQueryとプラグイン(jquery.ndla.person.date.js)を読み込みます。

<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js"></script>
<script type="text/javascript" src="jquery.ndla.person.date.js"></script>

検索結果を表示させたい要素のidを設定して、オプションの設定をしてください。birthが生年、deathが没年です。

<script type="text/javascript">
  $(document).ready(function(){
    // 例）sample02というid要素に2000年が生年の人名を表示
	$("#sample02").getNdlaFromDate({birth:2000});
    // 例）sample03というid要素に2012年が没年の人名を表示
	$("#sample03").getNdlaFromDate({death:2012});	
  });
</script>

最大100件まで取得します。

2000年が生年の人名

2012年が没年の人名
